# kursach-vsh-Development-of-an-online-veterinary-clinic-management-platform-
Цель проекта
Разработать web-приложение для автоматизации ключевых процессов ветеринарной клиники: учёт пациентов (питомцев), запись на приём, работа персонала с расписанием и медицинскими картами, формирование отчётов.
Платформа предполагает разграничение ролей пользователей и обмен данными с сервером через REST API.
